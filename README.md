# Web Application Security Labs

This repository also includes additional security labs covering common web vulnerabilities.

* DVWA exploitation
* JWT authentication vulnerabilities
* Tomcat penetration testing
* WordPress enumeration using WPScan
* PHP disabled functions bypass

---

# Methodology

The following methodology is applied during assessments:

1. Reconnaissance
2. Service enumeration
3. Vulnerability identification
4. Exploitation
5. Privilege escalation
6. Post-exploitation
7. Reporting

---

# Tools Used

Common tools used during the assessments include:

* Nmap
* Burp Suite
* BloodHound
* Impacket
* LinPEAS / WinPEAS
* Hashcat
* Netcat
* Gobuster / Feroxbuster
